# Royal Fashions – Modern E-commerce Frontend

Welcome to **Royal Fashions**, a sleek and responsive e-commerce frontend built with **HTML, CSS, and JavaScript**. This project blends elegant design with practical UI components to simulate a real-world fashion store experience.

##  Live Preview

---

##  Project Highlights

-  **Custom Branding**: Features a regal lion-head logo with crown for strong visual identity  
-  **Product Showcase**: Stylish cards for dresses and fashion items with hover effects  
-  **Coupon & Promo Integration**: Frontend logic for displaying discounts and offers  
-  **Responsive Design**: Optimized for mobile, tablet, and desktop views  
-  **Clean Codebase**: Modular HTML/CSS structure for easy scalability  
-  **Call-to-Action Buttons**: “SHOP NOW” and navigation links for user engagement
